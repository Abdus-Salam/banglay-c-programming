# বাংলায় সি প্রোগ্রামিং

# ওপেন সোর্স
এই বইটি সম্পুর্ন ওপেন সোর্স এবং স্বেচ্ছাশ্রমের জায়গা থেকে লেখা। 

# সূচনা
সি একটি স্ট্রাকচার প্রোগ্রামিং ল্যাংগুয়েজ।
